# CUNEF-Resources

Repository for different resources for CUNEF teachers. Datasets, web applications, notebooks, etc.


## `streamlit_apps`

In this section you can find different web applications developed for CUNEF courses:

- `bets_streamlit_app.`: Web application that simulates an optimization problem by a betting strategy.
- `salesman_streamlit_app.`: Web application that shows graphically the Traveling Salesman Problem in 2D.
