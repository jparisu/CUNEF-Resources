# CUNEF-Resources

Repository for different resources for CUNEF teachers. Datasets, web applications, notebooks, etc.


## `streamlit_apps`

In this section you can find different web applications developed for CUNEF courses:

### Bets

This app is a small interactive demo of a **betting game** where the user must find the best strategy to obtain the highest possible return.
You can find more information about this app in the [bets readme](./streamlit_apps/bets/bets_readme.md).

This application is online at: <https://cunef-resources-bets.streamlit.app/>


### Salesman Problem

This app is a small interactive demo of the **Traveling Salesman Problem (TSP)** and simple optimization methods.
You can find more information about this app in the [salesman readme](./streamlit_apps/salesman/salesman_readme.md).

This application is online at: <https://cunef-resources-salesman.streamlit.app/>
